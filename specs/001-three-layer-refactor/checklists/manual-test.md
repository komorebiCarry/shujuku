# 人工测试检查清单

每个阶段完成自动化验证（`verify-build.sh` + `tsc --noEmit`）后，按本清单在 SillyTavern 中逐项验证。

**测试方法**：将 `dist/index.bundle.js` 安装为 Tampermonkey 脚本，打开 SillyTavern，选择一个有聊天记录的角色卡。

---

## 阶段 1 + 2（初始化 + 基础）

仅需自动化验证，无需人工测试。`verify-build.sh` 确认产物与基线逐字节一致即可。

---

- [x] MT-01 脚本加载：安装产物后刷新 SillyTavern 页面，控制台无 ReferenceError / TypeError
- [x] MT-02 菜单入口：扩展菜单中出现「星·数据库 III」菜单项，点击可打开主弹窗

- [x] MT-04 主弹窗 - AI指令预设页：可切换预设，段落编辑器可添加/删除/编辑段落
- [x] MT-05 主弹窗 - API&连接页：API 模式下拉可切换（酒馆/自定义/直连），连接参数可编辑
- [x] MT-06 主弹窗 - 世界书页：世界书列表加载正常，条目勾选状态可切换

## 使用说明

1. 每个阶段 AI 完成自动化验证后，会提示你执行本清单中对应阶段的测试项
2. 逐项执行，通过的打 `[x]`，失败的记录现象
3. **全部通过**才能进入下一阶段
4. **任何一项失败**，停止并反馈给 AI 排查
